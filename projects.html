RCOS Collaboration

This semester we are collaborating with the Rensselaer Polytechnic Institute near Albany, New York. We are bringing one of their open source projects called YACS (Yet Another Course Scheduler) to NYU. Currently the NYU team is working on converting the Albert data to “YACS form”. Along with that there is work to be done on frontend and backend so YACS can be adopted for NYU. The team is looking for more members. If you would like to get involved please email bradley.brecher@nyu.edu.

Raspberry Pi Timelapse

Using a Raspberry Pi 3 and camera module (Camera Video Module for Raspberry Pi Model A/B/B+, Pi 2 and Raspberry Pi 3), we created a sunset timelapse.

Django Web App

Alexa With RaspberryPi

Using open source software to set up a Raspberry Pi to work as a smart speaker utilizing Amazon's AI, Alexa, and developing open source Alexa Skills to be used on the Raspberry Pi smart speaker.

Raspberry Pi Webserver

A webserver can refer to either software, hardware or a combination of the two working in unison. A web server is a computer that stores a websites in the form of documents and data, including HTML files, images, videos, etc and renders them to users when they are requested.
